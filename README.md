# ⚙️ Key Features

> [!WARNING]
> The add-on is still in Alpha. Expect Instabilities, breaking changes and modifiers incompatibilities when mixing versions.

### 📐 Model with NURBS
Through a growing set of GeometryNodes modifiers, SurfacePsycho allows true **non-destructive** NURBS modeling with real time editing ! You can trim, project, interpolate, and connect surfaces and curves with continuity control.

>  [!IMPORTANT]
> Anything you make with SP is strictly Blender-vanilla and the add-on is not necessary to share or re-open your files (the add-on is only needed for I/O and better UX)

### ⬆️ Export to CAD
Share your creation with your local polygon-allergic engineering department by exporting proper .STEP
In addition you can export any 2D shape as .SVG for a complete integration of the design process.

### ⬇️ Import CAD file
SurfacePsycho is close to able to do complete STEP/IGES import, and for the first time ever in any software I know, importing a .STEP gives you direct editing access to the shapes : No more dead surfaces !
